## Module Description
This Functions and Algorithms module is classroom-tested from Data 6 Fall 2025. 

In this module, students explore and develop functions, manage variable scope, and implement control structures such as nested conditionals. Students gain experience with scoping in Python as well as iterative processes using *for* and *while* loops for data transformations across strings, arrays, and tables. Students practice using boolean logic and use truth tables, short-circuiting and modulo operators. Students analyze validity and reliability, by exploring Body Mass Index (BMI) and the federal poverty line.

Throughout the module students will learn the following content topics: 

**Functional Programming, Scoping, and Iteration**

* Distinction between return and print values.  
* Mechanics of scoping in Python.  
* Using .apply().  
* Understanding the output of nested functions/loops.  
* Using for and while loops with arrays, and strings.  
* Performing data transformation and manipulations with slicing, concatenation and arrays.  
* Using NumPy tools like np.all() and np.any().

**Logic and Conditional Control Flow**

* Truth Tables.  
* Boolean Comparison.  
* Utilizing Boolean short-circuiting.  
* Using Modulo operators and boolean predicates.  
* Building functions to compute tiered data, such as tax brackets or health classification indices.

### **Measurement, Validity, and Reliability**

* Define and identify validity and reliability.  
* Examine the validity and reliability of BMI and Federal Poverty Line.   
* Assess how healthcare and legal thresholds affect individual risk assessments.
