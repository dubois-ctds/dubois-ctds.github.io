## Module Description
This Current Topics module is classroom-tested from Data 6 Fall 2025. 

In this module, students explore how data is collected, structured, and used to draw conclusions about the world. Beginning with measurements and Python dictionaries, students build toward web scraping and API-based data collection.


Throughout the module students will learn the following content topics:   
**Measurements, Validity, and Reliability:**

* Algorithms as tools for producing measurements  
* Operationalization of concepts into variables  
* Validity and Reliability

**File Formats and Dictionaries:**

* Common file formats: JSON, CSV, and HTML  
* Python dictionaries: key-value pairs, creation, access, and modification  
* Dictionary iteration using .keys(), .values(), and .items()  
* Nested dictionaries and working with JSON data

**Web Scraping and Data Collection:**

* HTML structure and how web pages are organized  
* Web scraping using the BeautifulSoup library and extracting HTML tags, attributes, and classes  
* Using the requests library to retrieve web data  
* APIs and RESTful vs. Python library-based API access
